## 수정 필요 사항
- 탭 바 (캘린더, 탐색, 루틴, my)

Home
- 제목 수정 가능 (입력)
- 랜덤 명언
- 일년의 몇 % 지났습니다

Routine
- 동그라미 가로 정렬 이미지 아이콘
- 루틴 추가 or 삭제
- 디자인 - 오늘 할 일 원 그래프

과제
- firebase 연결
